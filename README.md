# 🛡️ Tank Simulation Project

A multi-language demo project inspired by military tank systems and simulations.

## Features
- Tank movement simulation (Python)
- Battle controller (JavaScript)
- Web interface (HTML/CSS)
- Simple API server (Node.js)
- Physics engine (C++)
- Tank model (Java)
- Fuel calculator (Go)
- Diagnostics tool (Rust)
- Configuration files (JSON & YAML)

Power, armor, and mobility — simulated with code.
